# Cobblemon Unleashed

A Cobblemon adventure built on FTB Evolution — catch, battle, and breed Pokémon while exploring a tech-and-magic kitchen-sink modpack.

## Install

Download the latest release: [Releases page](../../releases/latest)

Full install instructions are in `INSTALL.md` inside the release zip.

## What's Inside

The Cobblemon ecosystem (Cobblemon 1.7.3, Mega Showdown, Battle Tower, Battle Extras, Legendary Monuments, Cobbledex, Cobblemon Quests Reloaded, AllTheMons x Mega Showdown), built on top of FTB Evolution's tech and magic foundation. BSL Shaders pre-configured. Distant Horizons for extreme view distances. Performance-tuned with Lithium, ModernFix, FerriteCore.

## Notes

Built on FTB Evolution. All mods retain their original licenses; this is an unofficial private assembly for friends-only play. Not for public redistribution.
